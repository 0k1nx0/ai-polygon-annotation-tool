# PolyAnnot v2.0 � AI Polygon Annotation Tool

AI-powered image annotation tool with polygon and bounding box support.

## Features
- Auto polygon segmentation (YOLOv8 + SAM)
- Bounding box detection
- Annotation review (thumbs up/down)
- Polygon/bbox correction
- Undo / Redo
- COCO JSON export
- Training dataset with SQLite
